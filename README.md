# Blockchain-Based Transcript Issuance and Verification System

## Overview
This project is a full-stack blockchain-integrated system that enables universities to issue tamper-proof academic transcripts and allows employers or institutions to verify transcript authenticity instantly.

The system stores transcript data securely in a relational database while storing only the cryptographic hash on the Ethereum blockchain to ensure immutability and trust.

---

## Tech Stack

Frontend: React.js, Tailwind CSS  
Backend: Spring Boot (Java), REST APIs  
Database: MySQL  
Blockchain: Ethereum (Smart Contracts using Solidity)  
Tools: Docker, Hardhat, GitHub

---

## Key Features

- Secure transcript issuance by university admin
- Blockchain-based transcript hash storage
- Student transcript access portal
- Public transcript verification portal
- QR-based verification support
- Role-based authentication system
- RESTful backend architecture
